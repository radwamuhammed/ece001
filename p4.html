  • Main levels of programming languages
  • Machine language:

    It is a series of zeros and units divided into instructions according to the processor. More precisely,the machine language may take the form of a hexadecimal coded code that at the hardware level can be understood and converted to binary code, i.e. zeros and units which the hardware can understand and implement directly.

  • Assembly language:

    It is a text language consisting of a specific number of instructions written in English that are converted to the machine language through the use of the assembly.

  • High Level Languages:

    These are text languages consisting of a number of rules and texts that can be used to write different programs. These languages offer great capabilities in terms of the ease of writing programs thanks to their texts most understandable to humans. This type of language is understood by converting it into machine language using translators.
